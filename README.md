# Отслеживание фандинга на биржах 

- собирает данные о ставках фандинга по биржам из config.yaml
- пишет полученные данные в БД
- на фронте выводит данные в виде таблицы с ворможностью сортировки

### Требования
- Go 1.23+
- Браузер (любой современный)

### Установка и запуск

```bash
git clone https://github.com/fiensola/funding.git
cd funding
make dev
